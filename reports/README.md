# Reportes del sistema SIGIM

Esta carpeta contiene reportes de ejemplo y evidencias generadas a partir de la base de datos del sistema.

## Archivos

- `informe_inventario_general.md`: resumen ejecutivo del estado del inventario.
- `informe_movimientos_recientes.md`: movimientos de inventario más recientes.
- `informe_alertas_ordenes.md`: alertas activas/atendidas y órdenes de compra sugeridas.

## Fuente de datos

Los reportes se construyeron con la base PostgreSQL del proyecto (`sigim_db`) y la estructura definida en el backend Django.

## Sugerencia

Si luego necesitas entregarlos en PDF, abre cada archivo Markdown y expórtalo desde tu editor o conviértelo con tu herramienta preferida.
